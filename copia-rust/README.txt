O objetivo deste projeto é criar uma cópia, mais parecida possível, com o site oficial em Português da RUST (uma linguagem de programação).
A ideia é criar a cópia sem olhar o source da página original. Apenas observando suas características e seu comportamento.

Isto serve apenas como um aprendizado (estou aprendendo sobre HTML, CSS, JS e desenvolvimento Web no geral). Não possui nenhum fim comercial.

Página orignal: https://www.rust-lang.org/pt-BR
